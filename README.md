# QQ-Group-Job
##解决的问题:
	大部分群成员不需要招聘信息,但是又有一部分成员有需求,所以能不能发招聘是群主不好抉择的问题.

##具体思路:
- 由群管理员维护一个应用入口(应用地址),引导普通用户与HR两类人前往应用.
- 普通用户拥有,注册,登录,及更新个人简历的功能
- HR可以查看用户的个人简历
- 管理员可以对普通用户进行技能评级(暂未实现)

##所用框架:
- 后端:Django
- 前端:AngularJS
